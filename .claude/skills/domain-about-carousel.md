# Domain — About Page Carousel

Auto-rotating testimonials carousel on `about.html`.

---

## Data Structure

Testimonials are stored as a hardcoded JS array:

```js
const testimonials = [
  { quote: "...", name: "...", title: "...", company: "..." },
  // 8 total entries
];
```

Array is defined in the inline `<script>` tag. To add or edit testimonials, modify this array directly.

---

## Rendering

| Step | Mechanism |
|---|---|
| Initial render | `showTestimonial(0)` called on page load |
| DOM target | Single `.testimonial-card` element; content replaced on each transition |
| Auto-advance | `setInterval(nextTestimonial, 8000)` — 8 second interval |
| Dots | `.t-dot` elements generated dynamically (one per testimonial); `.active` class tracks current |

---

## Transition

```
opacity: 0 (fade out, ~300ms) → update DOM content → opacity: 1 (fade in)
```

Implemented via JS: `card.style.opacity = '0'` → `setTimeout(() => { /* update + fade in */ }, 300)`.

---

## Manual Navigation

```js
// Clicking a dot:
dots[i].addEventListener('click', () => {
  clearInterval(autoTimer);   // stop auto-rotate
  showTestimonial(i);
  autoTimer = setInterval(nextTestimonial, 8000);  // restart
});
```

Manual click stops and restarts the timer to prevent immediate auto-advance after dot click.

---

## Key Functions

| Function | Purpose |
|---|---|
| `showTestimonial(index)` | Renders testimonial at given index; updates active dot |
| `nextTestimonial()` | Increments index (wraps to 0), calls `showTestimonial` |

---

## Modifying Testimonials

- **Edit existing:** Change `quote`, `name`, `title`, or `company` in the array
- **Add new:** Append object to array; dot navigation auto-scales
- **Remove:** Splice from array; dot navigation auto-scales

No changes needed to HTML structure — DOM is fully generated by JS.
