⚡ Things worth knowing
1. Run stopped mid-way? Just click the bookmarklet again — it picks up exactly where it left off (session is preserved in sessionStorage, no need to start over).
2. Can't auto-fill a field? Common with iframes & complex inputs — just type the value yourself. Learn mode captures what you typed and the generated Playwright spec will know to fill it the same way next time.
3. Watching the recording? Use the floating panel: ▶ Play auto-runs, Step walks one CTA at a time, Export downloads the bundle, Rules overrides field values, Log shows what happened.

4 Generate replay

Playwright spec runs in CI. Blueprint Builder turns this bundle into a modular Playwright project. Console replay pastes into DevTools console of the start page and replays the flow live in your browser — no Playwright required.

▶ Run the Playwright spec from your project root
$ npx playwright test --ui
$ npx playwright test flow-replay-<site>.spec.ts --headed
$ npx playwright show-report
First command opens the Playwright UI mode (recommended — interactive watcher, time-travel, screenshots, network). Second runs the spec headed so you see the browser. Third opens the full HTML report from the last run. Full button reference is in the README.
