4 Generate replay

Playwright spec runs in CI. Console replay pastes into DevTools console of the start page and replays the flow live in your browser — no Playwright required.

▶ Run the Playwright spec from your project root
$ npx playwright test --ui
$ npx playwright test flow-replay-<site>.spec.ts --headed
$ npx playwright show-report
First command opens the Playwright UI mode (recommended — interactive watcher, time-travel, screenshots, network). Second runs the spec headed so you see the browser. Third opens the full HTML report from the last run.
