02 / QA Tool
QA Extractor Pro
Bookmarklet → JSON → Playwright  ·  ↗ Open in new tab

Streamlines test automation setup. Drag bookmarklet to toolbar, click on any page to extract JSON, paste in tool, preview test steps in browser, then generate or download Playwright .spec.ts files. Integrates with npx playwright test for real execution.

03 / QA Tool
QA Dynamic Engine
Bookmarklet → JSON → Mega Test Suite → Deep Analysis  ·  ↗ Open full tool in new tab

Full-cycle form QA engine. Extract form fields from any page with the bookmarklet, generate a mega console test suite covering validation, accessibility, security (XSS/SQLi), and happy-flow — then paste results back for deep analysis with pass/fail stats, filtering, manual overrides, and Excel export.
