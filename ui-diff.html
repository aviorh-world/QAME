How to use
Step 01
Export from Figma
  1. Open screen in Figma
  2. Select the main Frame
  3. Run "Figma to JSON" plugin
  4. Copy the exported JSON
Step 02
Export from Browser
  1. Open the screen in browser
  2. Open DevTools console
  3. Paste & run the console script
  4. Save the downloaded JSON
Step 03
Run the Diff
  1. Paste Figma JSON below
  2. Paste Computed JSON below
  3. Click Run Deterministic Diff
  4. Review results
